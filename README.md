# Odin Recipes

First project of [The Odin Project](https://www.theodinproject.com/).

Project is about creating a recipe book.

Project showcase usage of following HTML tags:
* `<a>` - anchor
* `<h1>`, `<h2>` - headings
* `<ul>`, `<ol>`, `<li>` - unordered, ordered lists and list items
